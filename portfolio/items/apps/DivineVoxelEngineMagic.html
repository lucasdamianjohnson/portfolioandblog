Divine Voxel Engine Magic

About

Divine Voxel Engine Magic is a library for Divine Voxel Engine for rendering Magic Voxel scenes. It can parse Magic Voxel files into useful data for the engine.

It is also the first library to support the experimental new renderer for Divine Voxel Engine called the Quantum Renderer. The Quantum Renderer is a custom web gpu based renderer that uses compute shaders to render scenes with ray marching.
